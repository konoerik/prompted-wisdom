Prompted Wisdom

Welcome to Prompted Wisdom

The ideas that matter most — how to live, how to find meaning, how to face loss — have been explored by thinkers across every era and culture. We gave the same carefully constructed prompt to several leading AI models and asked each one to distill those ideas across the themes of human life. Same prompt, identical conditions, each model's response laid out for comparison.

The chapters are short enough to read in a sitting, and each one points back to the original sources if you want to go further. That's the real goal — not to replace those works, but to make them feel approachable.

Same prompt, word for word, across every model. Temperature set to zero. Every response stored with full provenance so the work can be verified or reproduced. Full methodology →

From the author

Foreword

The inspiration for this project came from a recent episode of Solved; Mark and Drew evaluated a large number of studies with respect to self-help and self-improvement techniques and then ranked them from worst to best in terms of what has been quantitatively proven to have a measurable and lasting effect.

Alongside the commentary of the podcast of humoristically tracing everything back to Aristotle (and a few other usual suspects), it got me thinking: what if I could capture those prominent thinkers and their ideas in a way that it is approachable to everyone. I would be the first to make use of this resource, but hopefully it could reach others too and serve as a gateway to the original works.

With access to LLMs, this sounded feasible all of the sudden. Not only that, but it expands the question to: what does each model think about these overarching and timeless themes? That made it for a bit more interesting project; rather than creating a little wiki for myself essentially, I could compare the output of different models and gather them all in one place not only for comparison, but as alternatives with (presumably) more or less the same output. This in turn required some methodology to be established, which I did.

The result is this present project, a short collection of twelve chapters across four leading models. See below for more details, read the chapters and check out the resources. Thanks for stopping by!

The project

Prompted Wisdom asks what happens when you give the same question to every major AI model. We took the themes that have occupied thinkers across every era and culture — how to live, what to value, how to face loss — and asked several leading models to distill them, under identical conditions. Then compared the results.

The methodology

Same prompt, temperature zero, per-model token limits, no system prompts — one canonical run per model per chapter, stored with full provenance. Full methodology and prompt →

Limitations and caveats

  • All API calls are routed through OpenRouter. Conditions are as identical as that layer allows — not a guarantee of parity with each model's native API.
  • Temperature zero is not deterministic in practice. Running the same prompt twice produces different output across all four models via OpenRouter. Each chapter is a single canonical run; divergence between runs is a known property of the pipeline, not an anomaly.
  • Models do not follow instructions uniformly. Structural markdown violations (stray headings, bold formatting) are stripped post-generation by a format script and logged in meta/format-log.json for transparency. Content-level issues — invented scenes, locations, or named individuals, and attributions that cannot be verified — are not corrected; they are properties of the model, not editorial choices.
  • This project was built with Claude Code. That makes it an inherently partial instrument in one direction, which is worth naming even if it cannot be corrected for. The tool is invaluable; the bias is real.

Author's Commentary

Notes from the reading.

This is an ongoing effort. As content is generated, reviewed, and revised across prompt versions, notes accumulate here.

Prompt v1.4 experiment

Added a full chapter list to the core persona to reduce cross-chapter repetition of thinkers and arguments. This version was not used for a full content generation — instead it became the trigger for a broader model evaluation that led to the phase 2 model upgrade.

Changes:

  • Core persona extended with a single sentence listing all twelve chapters and the instruction: "you are writing only the chapter you are given — do not preview, summarise, or defer to material that belongs in another chapter."

Outcome:

  • The chapter list worked clearly for Claude and meaningfully for GPT-5.4, producing distinct theses and better chapter separation. Effect was marginal for GPT-4o and Gemini 2.0 Flash — the structural problems in those models proved resistant to prompt-level fixes.
  • The test surfaced that GPT-4o and Gemini 2.0 Flash had reached their ceiling with the current prompt. This prompted a full evaluation of frontier model alternatives across seven candidates (Claude Sonnet 4.6, Claude Opus 4.6, GPT-5.4, Gemini 2.5 Pro, Mistral Large 3, and the existing models).
  • Decision: retire all four current models and regenerate with frontier equivalents. See ADR-6.

Notes:

  • Claude (Sonnet 4.5 → Opus 4.6): Opus produced the sharpest conceptual distinctions — "Happiness is a state. Meaning is a direction." and "Suffering is not redeemed by meaning. But meaning can be built from suffering, the way a wall can be built from broken stone." Chapter separation was clean across all pairs.
  • GPT-4o → GPT-5.4: GPT-4o's default textbook-survey shape ("a common thread emerges") persisted despite the chapter list. GPT-5.4 broke this pattern entirely — real theses, earned quotes, distinct closers per chapter.
  • Gemini 2.0 Flash → Gemini 2.5 Pro: Flash's deterministic opening pattern and near-identical structure did not respond to v1.4. Gemini 2.5 Pro (a thinking model) eliminated both issues — essays rather than structured summaries. Requires max_tokens raised to 8000 to clear the thinking token budget.
  • Mistral Large 2411 → Mistral Large 3: Large 2411 remained verbose and continued bleeding Frankl into unrelated chapters. Large 3 is cleaner, has real theses, and is 4× cheaper. No-brainer replacement.

Prompt v1.3b full generation

Two prompts were tried in sequence — v1.3a and then v1.3b — with the main goal of improving the content while removing the fabricated personal accounts.

Changes:

  • Removed the travel persona altogether and moved towards a reader/thinker narrator instead.

Outcome:

  • The risk here was ending up with cold, detached text — moving away from the traveller who would have had boots on the ground, so to speak. Surprisingly, that did not happen. The generated text was the best so far: broad inclusion of thinkers, philosophies, and ideas, while still maintaining some first-person statements without the exaggerated stories.

Notes:

  • Claude: Continues to be balanced.
  • GPT-4o: Following the prompt closely. Several chapters open with the same wording — "human tapestry" or "single/twin threads of existence." There is unmistakably a personal account at the second-to-last paragraph. It also keeps echoing "this is an anchor, not a map" in closing passages — lifted verbatim from the prompt itself, appearing in 7 of 12 chapters.
  • Gemini: Overall better than GPT, with a more diverse writing style that doesn't feel scripted. A leaning towards Ancient Greece in many openings, Buddhism a close second. The use of "of course" and "as usual" makes it feel preferential — admirable as the Greeks are, this sits oddly in a world where there are no absolutes.
  • Mistral: More in-depth content, sometimes spanning a couple of paragraphs on a single thinker before moving on, which gives it a continuity the others lack. On virtue it does an especially good job — nails Aristotle on the types of friendship and even provides a Buddhist analog. Some errors worth noting: "Confucius, in the Confucian tradition" (circular); and it placed the Stoics in the Himalayas, which is a clear hallucination. On the cardinal virtues, it credits the Greeks rather than the Stoics — this is actually historically accurate, as Plato laid them out first and the Stoics inherited them.

Prompt v1.3a rejected

An attempt to remove the witnessing framing while preserving the warmth of the travel persona.

Changes:

  • Removed the "witnessing" phrasing but maintained the travel persona.

Outcome:

  • This did not eliminate the strong personal accounts, particularly in Gemini.

Prompt v1.2 intermediate

Building up from v1.1, some more changes to steer the models into a more idea-inclusive and less hallucinatory terrain.

Changes:

  • The explicit names were removed and replaced with broad categories — Stoicism, Buddhism, and so on.
  • The persona added a "lines you have carried" clause to encourage direct quotation where applicable, instead of paraphrasing every time.

Outcome:

  • This helped broaden the range of thinkers and schools of thought significantly. Immanuel Kant was mentioned for the first time, and Jean-Paul Sartre had several entries.
  • The fabricated stories were still present, however, which disqualified this prompt version.

Prompt v1.1 intermediate

This was an iteration on v1.0 to improve what was included in terms of schools and thinkers.

Changes:

  • Aristotle and Plato were added by name, and the list was separated by semicolons to nudge towards eras and regions as opposed to a fixed list.
  • The chapter instruction was updated to "follow the ideas themselves" — pointing towards the continuity of thoughts, in the hopes of including not only the originator of an idea but also those who carried it forward.

Outcome:

  • This did not work as well as intended. Some differences, yes, but still somewhat rigid — now with the addition of Aristotle and Plato specifically.
  • Another thing noted (although not necessarily introduced in this version) is the lack of memorable quotes — something that would be important for a new reader especially.

Prompt v1.0 full generation

This was the first version of the prompt that was good enough for content generation. It was crafted around two main ideas: 1. Be inclusive in the thinkers, ideas, and schools of thought that have left their mark, and 2. Use the "walking philosopher" angle behind a journalist persona to give it a practical angle and a warm tone as opposed to an academic account. This was largely inspired by the book A Philosophy of Walking by Frédéric Gros.

What worked:

  • Coherent outputs for all models — hence the decision to publish.
  • Structured paragraphs in a way that were consumable by the reader.

What needed improvement:

  • By listing certain names in the prompt, the models largely interpreted the intention as "cover topics from these thinkers primarily," even though the phrasing specified this as a range, not a list.
  • Due to the above, sections felt a bit scripted. The content was still good, but you could blindly predict the next chapter would open with the Stoics (since they were mentioned first in the prompt).
  • The walking philosopher persona also backfired to varying degrees. Some models interpreted the intention as "always include a personal story," which ranged from "I met a woman in Nepal" to "I have witnessed loss and peril." This is a very strange thing to read, especially when you know it is synthesised text, and even more so on certain chapters like suffering and mortality.
  • A secondary side effect of the above was some models continuing to invent stories chapter after chapter. Keeping in mind each chapter is generated by a separate prompt, this is to be expected; but again, it is an eyesore when you know these are not actual experiences and therefore add no value.

Notes:

  • Claude seemed very measured — chapter after chapter, laying out the content following Stoicism, Buddhism, Confucianism, and Montaigne. Not much in terms of quotes or background, however, which a new reader would benefit from.
  • GPT and Gemini seemed very similar in their approaches — a bit overly warm (or more exaggerated) in how they followed the prompt, always including a "personal story." Very interesting how the story was always in Nepal, Marrakesh, or the Andes. Marrakesh especially came up too many times — some weighted training on The Alchemist, perhaps? (Just kidding.) They also seemed to have a singular phrasing for Nietzsche and Montaigne, which might turn away some readers. This is likely due to the fact that each prompt is generated atomically, so the models reach for the same few descriptions (radical, antithetical, etc.).
  • Mistral did something interesting: it introduced the thinkers by full name and some background, as opposed to a dry description. According to a Claude analysis it also misattributed a quote, which is not something we want.
