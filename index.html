Prompted Wisdom

The ideas in every self-help book you've ever read are not new — and that's the most reassuring thing about them.

The concepts that keep resurfacing — how to live with uncertainty, how to find meaning, how to face loss — were first worked out roughly 2,500 years ago, across Ancient Greece, India, and China. The Stoics, the Buddhists, the Confucians didn't know about each other, but they were asking the same questions and arriving at strikingly similar answers. Scholars have returned to this material for centuries. So have ordinary people, in difficult moments, looking for something solid to hold onto. Every self-improvement book, every thoughtful podcast, every philosophy course eventually leads back to the same wells.

Large language models were trained on vast amounts of human writing — including, almost certainly, the public domain works of the thinkers who shaped these ideas. Plato, Marcus Aurelius, the Upanishads, Montaigne. When you ask a model to distill this material, something interesting happens. The results aren't coherent because the model is wise. They're coherent because the source material is, and because it has been refined over millennia.

This is a small experiment and an open invitation. We gave the same carefully constructed prompt to several leading models, under identical conditions, and asked each one to write a concise guide across ten themes — the kind of overview you could actually read and carry with you. Concise by design: easier to read, and a fair test across models. The chapters are short enough to explore in a sitting, and each one points back to the original sources if you want to go further. That's the real goal — not to replace those works, but to make them feel approachable.

About

A small experiment in ancient wisdom and modern AI.

The project

Prompted Wisdom is an open experiment. The goal is simple: take the core themes of human wisdom — the ones that have been written about for 2,500 years — and ask several leading AI models to distill them, under identical conditions, into something readable. Then compare the results. The chapters are intentionally concise: short enough to read in a sitting, specific enough to point you toward the original sources.

The methodology

Every chapter is generated from the same prompt, word for word, across every model. Temperature is set to zero to minimise variation within a model. Token limits are fixed so no model wins by writing more. No system prompts, no extra instructions — one clean input. Each response is stored with full metadata: model version, generation timestamp, token counts, and a SHA-256 hash of the raw output so the content can be verified against what the model actually produced.

The full prompt and all API parameters are documented in PROMPT.md in the GitHub repository.

The sources

Every resource linked from this site is free to access, distraction-free, and points directly to the text — no paywalls, no algorithmic feeds, no recommendation engines. Primary sources come from Standard Ebooks, Project Gutenberg, Internet Archive, Wikisource, and LibriVox. Reference entries come from the Stanford Encyclopedia of Philosophy.
